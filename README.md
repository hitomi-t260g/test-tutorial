## 📝 About

作ってみたユニットテストを蓄積するリポジトリです。
現状は mocha によるユニットテストのみ test フォルダに入っています。

## 🏃 How to run

node および npm がインストールされている状態で mocha および chai をインストールしておきます。
インストールは以下。

```
npm install mocha --save-dev
npm install chai --save-dev
```

テストは、以下をターミナルで実行すると動きます。

```
npm test
```
